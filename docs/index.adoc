Technische Spezifikation für Planung, Bau und Betrieb von Schmalspur-Bahnen.

SBO ist eine private, offene technische Spezifikation für Schmalspur-Bahnen (Garten- und Parkbahnen).
Es definiert Klassensysteme, technische Anforderungen, Berechnungsmethoden und Sicherheitsrichtlinien.

== Aufbau der Spezifikation

Diese Spezifikation ist modular aufgebaut:

* `core/` – Spurweitenunabhängige Prinzipien, Definitionen und Berechnungsmethoden
* `specs/sbo-XXX/` – Spurweitenspezifische Parameter und Grenzwerte

Die Core-Kapitel werden per `include::` in die jeweilige Spurweiten-Spezifikation eingebunden.
Endanwender lesen immer die fertige Spurweiten-Spec (z. B. SBO-287), nicht die Core-Dateien separat.
