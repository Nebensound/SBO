== Zusammenfassung

SBO-Core definiert die spurweitenunabhängigen Prinzipien, Klassensysteme, Rechenmodelle und Sicherheitsphilosophie für Schmalspur-Bahnen. Konkrete Zahlenwerte werden in den spurweitenspezifischen Specs (z. B. SBO-287) festgelegt.
