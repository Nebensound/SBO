= Fahrzeuge

[abstract]
Dieses Kapitel definiert die Anforderungen an Fahrzeuge: Wagen, Lokomotiven, Personenfahrzeuge, Schwerpunkt, Kupplungen.

== Allgemeine Fahrzeuganforderungen

_Platzhalter: Lichtraumprofil, Spurkranz, Puffer, Sicherheitseinrichtungen_

== Wagen

_Platzhalter: Güterwagen, offene/geschlossene Bauarten, Ladungssicherung_

== Lokomotiven / Triebfahrzeuge

_Platzhalter: Adhäsionsmasse, Antriebsart, Zugkraft_

== Personenfahrzeuge

_Platzhalter: Sitze, Türen, Notausstiege, Sicherheitsgurte_

== Schwerpunkt und Lastverteilung

_Platzhalter: Schwerpunkthöhe, Kippstabilität, Lastverteilung auf Achsen_

== Kupplungen und Zug-/Druckkräfte

_Platzhalter: Kupplungstyp, maximale Zug-/Druckkraft, Kuppelhöhe_
